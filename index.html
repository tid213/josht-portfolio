Green Capz

Introducing Green Capz, a sophisticated web application crafted using MongoDB, Express, React, and Node (MERN stack). Designed specifically for hydroponic enthusiasts, this innovative platform seamlessly captures sensor data sent from the Green Capz controller. Users are granted real-time access to both current and historical readings of their hydroponic garden.

With its intuitive interface and robust functionality, Green Capz empowers users to monitor crucial parameters, ensuring optimal conditions for their plants. Experience precision and convenience in hydroponic gardening with Green Capz, your gateway to data-driven cultivation.

Green Notez

Introducing Green Notez, an intuitive application I developed to monitor hydroponic gardening data effortlessly. This user-friendly app simplifies the process by focusing on essential aspects such as water changes, daily checks, and addressing problems. Unlike other intricate grow tracking apps, Green Notez prioritizes recording key data points. Users can easily share this information through visually appealing .jpg note cards.

Built with React and Supabase for seamless database storage and user authentication, Green Notez offers a streamlined experience. It allows users to efficiently document and monitor their hydroponic garden, ensuring a hassle-free and insightful gardening journey.

Green Capz Sensor Controller

In the development of the Green Capz application, I engineered a sophisticated sensor controller by seamlessly integrating a Raspberry Pi and Arduino board. The Arduino board adeptly reads data from analog sensors, converting it into a digital signal, which is then transmitted to the Raspberry Pi. Operating on an Express/Node/MongoDB/Pug stack, the Raspberry Pi efficiently stores user data, ensuring seamless access for Green Capz users.

To enhance user experience, real-time readings are displayed through a mounted screen on the enclosure housing the Raspberry Pi and Arduino boards. While the Arduino board required minimal C code, this project served as a valuable opportunity for me to delve into a new programming language, significantly expanding my skillset in the process.
